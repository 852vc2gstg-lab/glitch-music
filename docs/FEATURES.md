# Ghxsty Music - Features

## Core playback
- Local library playback (IndexedDB-backed)
- Play / pause / prev / next / seek / volume
- Shuffle + repeat modes
- Queue / upcoming track behavior
- Fullscreen player mode

## Library and organization
- Tracks list with search/filter
- Favorites
- Pinned tracks
- Playlists (create/edit/delete)
- Add/remove track to playlists
- Bulk track edit (title/artist/album/cover)

## Metadata and visuals
- Cover artwork resolution and fallback logic
- Album and genre metadata enrichment
- Dominant-color based UI theming
- Artist info and artist profile sections
- Lyrics fetching and caching

## Online/music sources
- YouTube / YouTube Music search flow
- Link-based download/import flow
- Artist albums/singles/top songs sections (YT Music-backed)

## UX/system
- First-run wizard (theme/language/close behavior)
- Dependency notice modal for missing tools
- Notifications center
- Download jobs panel
- Stats panel
- Settings panel (audio/theme/system/cache/etc.)

## Update/distribution
- In-app update check via GitHub Releases
- Update actions: Check / Download / Restart and install
- NSIS packaging (Windows) and Linux packaging scripts

## Integrations
- Discord Rich Presence
- Error report send flow (Discord webhook)
