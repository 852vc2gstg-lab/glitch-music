# GLITCH Music 

<img src="https://github.com/ghxsty-dev/glitch-music/blob/main/.github/GLITCHyazi%20(800%20x%20200%20piksel).png?raw=true&" width="1200" />

GLITCH Music, klasik bir oynatıcıdan daha çok “kişisel müzik merkezi” gibi çalışıyor: yerel kütüphane + YouTube Music arama/indirme + playlist yönetimi + gelişmiş ve modern player + lyrics (zaman ayarlı) + sanatçı/albüm keşfi tek uygulamada birleşiyor. Üstelik çok detaylı kişiselleştirme (tema, şeffaf mod, performans, kısayollar, davranış ayarları) ve masaüstü odaklı akıcı bir arayüz var.

Rakiplerden ayrılan net noktalar da şu: Discord Rich Presence entegrasyonu, uygulama içi gelişmiş indirme/aktarım akışları, aylık özet/istatistik yaklaşımı, offline-first mantık ve senin kullanımına göre şekillenen “öneri + keşif” deneyimi. Yani sadece müzik çalan bir app değil, düzenleyen, keşfettiren ve kişiselleşen bir ekosistem.

## Nasıl kurulur?

- Uygulamayı [GLITCH Music Setup](https://github.com/ghxsty-dev/glitch-music/releases)'tan kurduktan sonra kurulum ekranından kurulur. Eksik modüller için komutlar:

### Windows:
```bash
winget install python
winget install ffmpeg
pip install ytmusicapi
winget install yt-dlp
```

### Linux
```bash
Yakında
```

## Nasıl kullanılır?

- Uygulamayı kurduktan sonra sol üstte bulunan Oluştur tuşuyla çalma listesi oluşturabilirsiniz. TXT ile aktar ile Playlist aktarabilirsiniz. YouTube Music çalma listelerini aktarmak için Sağ üstteki ayarlar sekmesinden Ortak kaynak bölümüne gelin. Google hesabınızı bağlayın ve playlistlerinizi aktarın.

- Bir hata bulmanız veya bizle iletişime geçmek istemeniz durumunda [buraya](https://discord.gg/qc7t8xhUC) tıklayıp Discord sunucumuzdan bize ulaşabilirsiniz.
